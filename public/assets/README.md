# Cat Paw Asset

Please add your cat paw GIF file here with the name: `cat-paw.gif`

The GIF should match the animated cat paw from the Figma design.

**Required:**
- Filename: `cat-paw.gif`
- Location: `public/assets/cat-paw.gif`
- Format: GIF (animated)
- Recommended size: 128x128px or larger

Once added, the application will display the animated cat paw mascot with the "Best of Luck!" speech bubble.
